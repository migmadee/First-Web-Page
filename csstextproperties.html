Introduction to CSS Text Properties

CSS provides several properties for styling and formating text.
This tutorial covers the most commonly used text including color,alignment,decoration,transformation,spacing, and shadow effects.

Text Color

The color property is used to set the color of text.

This text is colored using the hex code#6600.

CSS code: color:#ff6600;

Text Alignment

The text-align property is used to set the horizontal alignment od text.

This text is aligned tp the left(default)

CSS code: text-align: left; CSS code: text-align: justify;

Text Decoration

the text-decoration property is used to add or remove decorations from text.

This text has an underline.

CSS code: text-decoration: underline.

this text has overline;

CSS code: text-decoration:over line;

this text has line through

CSS code: text-decoration: line through;

Links typically have an underline by default,but it can be removed:

This line has no underline

CSS code:text-decoration: none;

Text transformation

The text-transform property is used to change the capitilization of text.

this text is transformed to uppercase.

CSS code: text-transform: lowercase;

this text is transformed to lowercase;

this text is transformed to capitilize each word.

CSS code: text-transform: capitalization;

Text Spacing

CSS provides properties to control the spacing between letters word,and lines.

This paragraph demonstrates letter spacing, word spacing, and line height. Notice how the letters have more spcae between them, the words are further apart, and there's more vertical space between lines.

CSS code: letter-spacing; 2px; word-spacing:5px; line-height:2;

Text Shadow

This text-shadow property adds shadow effects to text.

This text has basic shadow effect.

CSS code: text-shadow: 2px 2px 4px #888

This text has a glow effect.

CSS code: text-shadow: 0 0 5px #(ff6600),0 0 10px #ff6600;
